# BD Paneling Studio (BDパネリング・スタジオ)

伝統的なバンド・デシネ（BD）特有の美しい大判サイズ感、間白（マージン）のリズム、そしてテキスト配置をブラウザ上でリアルタイムにシミュレート・検証できる、軽量なパネリング（コマ割り）Webアプリケーションです。

単一のHTMLファイルのみで構成されており、サーバーや外部ライブラリを一切使わず、オフラインのままでも即座に動作します。ネーム（絵コンテ）のプロトタイピングや、BDの画面構成の学習ツールとして最適です。

---

## ✨ 主な機能

### 1. 📐 伝統的なBD大判比率の完全再現
* **大判（フランド・フォルマ）比率:** 1ページあたり約 `1 : 1.35〜1.4` という、日本の一般的なコミック（B6判など）に比べて縦にスラッと長いBD独特のプロポーションを維持。
* **見開き対応:** 左右2ページを並べた際、美しい **「横 1.4 : 縦 1」** の比率で固定され、画面幅に合わせてレスポンシブに拡大縮小します。

### 2. 🎛️ 上下・左右の間白（Gutter）独立リアルタイム調整
* 視線誘導や段（Row）ごとの区切りを意識できるよう、**「上下の間白」と「左右の間白」を個別にピクセル単位で調整可能**なスライダーを搭載。
* 伝統的な「上下を広く、左右を狭く」したレイアウトから、モダンな「隙間のない密着型」まで自由にシミュレートできます。

### 3. 💬 コマ内テキスト入力 ＆ 演出スタイル切り替え
* 各パネル（コマ）を直接クリックするだけで、セリフやモノローグの文字入力が可能。
* マウスホバーで表示されるミニメニューから、演出に合わせた3つのスタイルにワンクリックで切り替えられます。
  * **標準:** コマ中央にシンプルにテキストを配置。
  * **吹き出し:** 文字を囲む簡易的な丸いバルーンを生成。
  * **ナレーション:** 古典的なBDでよく見られる、**「黄色い背景の四角いキャプションボックス」**に変化。文字が入った際の見開きの密度や可読性を視覚的に確認できます。

### 4. 🖱️ グリッドパターンのワンクリック切り替え
* 左ページ、右ページそれぞれで、BDの王道である**「4段×3列（計12コマ）」**と**「3段×4列（計12コマ）」**を個別に瞬時に切り替えられます。
* 切り替えても、各コマに入力したテキストの内容はそのまま保持されます。

### 5. 💻 フルスクリーン・デスクトップアプリ風レイアウト
* 画面の縦幅（`100vh`）を基準にすべてが自動計算され、不要なスクロールバーを排除。PCの画面全体を使った没入感のある作業環境を提供します。

---

## 🚀 使い方（クイックスタート）

1. このリポジトリにある `index.html` をローカル環境にダウンロードします。
2. ダウンロードした `index.html` を、お使いのブラウザ（Chrome, Safari, Edge, Firefoxなど）にドラッグ＆ドロップして開きます。
3. 外部通信やインストールの必要はなく、すぐに使用可能です。

---

## 🛠️ 技術スタック

* **HTML5** (セマンティックな構造定義)
* **CSS3** (CSS Gridによる緻密なコマ割り表現、Aspect-Ratioによる比率固定、レスポンシブデザイン)
* **JavaScript** (Vanila JS / 外部ライブラリ・フレームワーク不使用による軽量・高速なDOM操作)
